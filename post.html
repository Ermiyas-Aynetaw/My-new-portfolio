Project and Developer Integrator Website

The Project and Developer Integrator Website, is a Python Django-powered platform facilitating seamless connections between companies and skilled developers.

With a user-friendly interface, developers can easily create accounts and showcase their portfolios, while advanced search filters enable companies to find the perfect match.

A unique feature assesses developers' motivation and patience, adding depth to the recruitment process.

This project not only simplifies developer recruitment but also showcases my expertise in Django and innovative problem-solving.

Technologies i have used:
  • - Python
  • - Django
  • - Django REST Framework
  • - PostgreSQL
  • - Templates
